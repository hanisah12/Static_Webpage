HTML Advanced Questions & Answers

Advanced Level

  • Q1: What are custom data attributes?
    A: Attributes starting with data- to store custom data.
  • Q2: Difference between HTML5 localStorage and sessionStorage?
    A: localStorage persists, sessionStorage clears on tab close.
  • Q3: What is the purpose of <canvas>?
    A: For drawing graphics using JS.
  • Q4: What is the difference between inline SVG and <img> SVG?
    A: Inline SVG can be styled and animated, img SVG cannot.
  • Q5: What is Web Components?
    A: A way to create reusable custom elements.
  • Q6: Difference between defer and async in script?
    A: Async runs immediately, defer waits until DOM is ready.
  • Q7: What is accessibility in HTML?
    A: Making websites usable for disabled users (using ARIA roles).
  • Q8: Difference between HTML imports and modules?
    A: Imports load HTML, modules load JS in modern browsers.
  • Q9: What is the shadow DOM?
    A: Encapsulated DOM for custom elements.
  • Q10: What are progressive web apps (PWAs)?
    A: Web apps that behave like native apps.
