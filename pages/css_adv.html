CSS Advanced Questions & Answers

Advanced Level

  • Q1: What are CSS variables?
    A: Custom properties defined with --var syntax.
  • Q2: Difference between absolute, relative, fixed, and sticky positioning?
    A: Sticky toggles between relative and fixed depending on scroll.
  • Q3: What is the difference between transitions and animations?
    A: Transitions occur between states, animations run continuously with keyframes.
  • Q4: What are CSS preprocessors?
    A: Tools like SASS/LESS that extend CSS.
  • Q5: What is the difference between inline SVG styling and CSS styling?
    A: Inline SVG allows more control, external CSS has limited effect.
  • Q6: What is responsive design?
    A: Designing websites to work on all screen sizes.
  • Q7: What are media queries?
    A: CSS rules applied only on specific screen sizes.
  • Q8: What are CSS masks and clipping?
    A: Techniques to hide parts of elements using shapes or images.
  • Q9: What is the difference between transform and translate?
    A: Translate moves elements, transform includes rotate, scale, skew, etc.
  • Q10: What are CSS custom fonts?
    A: Fonts imported using @font-face or Google Fonts.
